# LLM Zoomcamp 2026

This repository contains homework solutions and the capstone project for the [LLM Zoomcamp 2026](https://github.com/DataTalksClub/llm-zoomcamp) course by DataTalks.Club.

## Course Structure

- [Module 1: Agentic RAG](./01-agentic-rag/) - Build a RAG pipeline with keyword search and make it agentic with function calling
- [Module 2: Vector Search](./02-vector-search/) - Semantic search with embeddings using minsearch, sqlitesearch and PGVector
- [Module 3: Orchestration](./03-orchestration/) - AI orchestration with Kestra
- [Module 4: Evaluation](./04-evaluation/) - Measure retrieval and answer quality, offline and online
- [Workshop: dlt](./workshops/dlt/) - Pull LLM traces from a monitoring service for analytics with dlt
- [Module 5: Monitoring](./05-monitoring/) - Monitor user feedback and system health with live dashboards
- [Module 6: Best Practices](./06-best-practices/) - LangChain, hybrid search and reranking
- [Module 7: Project Example](./07-project-example/) - End-to-end project: a fitness assistant built with LLMs
- [Capstone Project](https://github.com/DataTalksClub/llm-zoomcamp/blob/main/project.md) - Ship a complete end-to-end RAG application of your choice

- Python
- OpenAI API (GPT-4o mini, text-embedding models)

- SQLite / sqlitesearch
- PostgreSQL with PGVector
- Qdrant (vector search)
- Kestra (orchestration)
- dlt (data ingestion)
- LangChain
- Docker
- Streamlit / FastAPI (application layer)
- Ragas / LLM-as-a-Judge (evaluation)
- Grafana (monitoring)
